# Supply Chain Network Architecture & Cost Optimization Engine

## 🎯 Project Objective
This case study models an end-to-end enterprise logistics data pipeline to identify financial overhead leakage and delivery SLA bottlenecks across regional manufacturing nodes.

## 🛠️ Tech Stack & Skills
* **Database Layer:** MySQL (Multi-Stage CTEs, Non-Equi Joins, Window Functions, DENSE_RANK)
* **Presentation Layer:** Microsoft Excel (Dynamic Array Integration, Conditional Layouts, Executive Reporting)
* **Domain Framework:** Supply Chain Operations, Operations Audit, Cost Reallocation

## 🔍 Core Strategic Takeaways
* **Financial Footprint:** Isolated system-wide financial leakage, proving that a concentrated minority of warehouses consume a disproportionate share of global logistics expenditure.
* **Bottleneck Identification:** Cross-referenced transactional order frequencies against fixed factory capacity limits to map asset over-utilization zones.
* **SLA Resolution:** Correlated capacity constraints directly with customer delivery delays, providing data-backed evidence to guide corporate resource shifting.
